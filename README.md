# Виктория | Эксперт по продвижению личных брендов

**Веб-сайт-портфолио для эксперта по созданию и продвижению личных брендов**  
*Аутентичность как конкурентное преимущество*

---

## О проекте

Этот сайт представляет профессиональное портфолио **Виктории Пугач** — эксперта по стратегическому PR и продвижению личных брендов. Проект сочетает в себе элегантный дизайн, современные веб-технологии и полную адаптивность для всех устройств.

###  Ключевые особенности:
- **Чистый минималистичный дизайн** в темной цветовой гамме с акцентами
- **Полная адаптивность** — от десктопа до мобильных устройств
- **Интерактивные элементы** с плавными анимациями
- **Оптимизированная производительность** и быстрая загрузка
- **Доступность** (accessibility) для всех пользователей

##  Возможности сайта

### Для посетителей:
-  **Адаптивный дизайн** — одинаково удобно на любом устройстве
-  **Видео-фон** на главном экране (автоматическое отключение на мобильных)
-  **Интерактивная карусель** с кейсами (свайп, автопрокрутка, клавиатурное управление)
-  **Отдельная страница** с политикой конфиденциальности
-  **Быстрые контакты** — телефон, email, социальные сети

### Для владельца:
-  **Портфолио услуг** в структурированном виде
-  **Демонстрация результатов** (статистика + кейсы)
-  **Блок отзывов** клиентов
-  **Готовая контактная информация**
-  **Юридическая защита** — политика конфиденциальности

##  Технологии

### Frontend:
- **HTML5** — семантическая разметка
- **CSS3** — кастомные свойства (CSS Variables), Flexbox, Grid, анимации
- **JavaScript (Vanilla)** — без зависимостей, чистый код
- **Адаптивная верстка** — 6 брейкпоинтов (от 350px до 1440px+)

### Оптимизация:
- **Ленивая загрузка** изображений и видео
- **Оптимизация анимаций** для мобильных устройств
- **Кэширование** ресурсов

### Инструменты разработки:
- **Git** — контроль версий
- **Google Fonts** — Cormorant Garamond, Inter
- **SVG-иконки** — векторная графика
- **Медиа-запросы** — адаптивные изображения

##  Адаптивность

| Устройство | Разрешение | Особенности |
|------------|------------|-------------|
| **Мобильные** | до 767px | Гамбургер-меню, отключены сложные анимации |
| **Планшеты** | 768px-1023px | Упрощенная верстка, оптимизированные карточки |
| **Ноутбуки** | 1024px-1439px | Полная функциональность |
| **Десктоп** | 1440px+ | Максимальный размер, улучшенная типографика |
